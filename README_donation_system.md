\# Donation Management System



A brief description of what this project does (e.g., managing donations, donors, and distribution using Spring Boot and MySQL).



\## 📋 Prerequisites

Before running this project, ensure you have the following installed:

\* \*\*Java JDK 17\*\* or higher

\* \*\*MySQL Server\*\*

\* \*\*Maven\*\* (optional, if using an IDE like IntelliJ or Eclipse)



\## ⚙️ Setup \& Installation



\### 1. Database Setup

1\. Open your MySQL Workbench or Command Line.

2\. Run the SQL script located at:

&#x20;  `src/main/resources/sql/DonationSystemSQL.sql`

3\. Update the database credentials in `src/main/resources/db.properties`:

&#x20;  ```properties

&#x20;  db.url=jdbc:mysql://localhost:3306/your\_db\_name

&#x20;  db.username=your\_username

&#x20;  db.password=your\_password

4\. Open Eclipse Import this 

5\. Right click your package

5\. go to maven update project and select okay.

6\. click project on top.

7\. select clean.

8\. select launcher.java and run.
